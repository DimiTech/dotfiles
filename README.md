## Vim

### Installing plugins and theming:

The following plug-ins are used:
```
pathogen
syntastic
powerline
nerdtree
ctrlp
auto-pairs
indentLine
vim-gitgutter
vim-fugitive
diffchar
silver
vim-ags
emmet-vim
```

The theme is called `codedark`, influenced by Visual Studio colors.
